baraction.py for tiling window managers, such as Spectrwm

Can be used by other window manager and application bars that accept a stdout text stream, such as i3status, xmobar, tmux, etc.

Addresses the repetitive process creation typical of baraction.sh scripts to ensure low latency and resource efficiency

Monitoring of CPU, RAM, storage, network (rx/tx), and uptime

Uses built-in python modules exclusively. No external packages necessary

Written in Python 3 (Versions 3.6+ ; Not backwards-compatible)
